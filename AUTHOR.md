## وصف المشروع / Project Description

**GeoData Hub** — منصة رقمية عربية متكاملة لإدارة واستعراض وتحميل البيانات الجغرافية المكانية.  
مبنية بأحدث تقنيات الويب (React 19, MapLibre GL JS, TypeScript 6, Tailwind CSS 4)  
وتعمل بمصادر بيانات مفتوحة بالكامل.

A comprehensive Arabic web GIS platform for managing, browsing, and downloading geospatial data.  
Built with modern web technologies and fully open data sources.
